# Changelog

## 1.0.0 - 2026-06-24

- Initial independent Go/Fyne implementation.
- Keyring, RSA generation, import, export, revocation and local trust.
- Text and file encryption/decryption, multiple recipients and password mode.
- Detached, inline and cleartext signing/verification.
- Native vault, session cache, authenticated backup and HKP/HKPS.
- CLI, drag and drop, file opening, package metadata and macOS integration documentation.
- Trust warnings, full/short Key ID preference and persistent backup reminder.
- Content-based routing for keys, messages and signatures opened by the operating system.
- Precise recipient selection during streaming decryption and safe fallback for stale vault secrets.
- Cryptographic, persistence, streaming, backup, transactional-write and Fyne rendering tests.
